# OpenSentinel

Autonomous Vulnerability Intelligence Platform built with Python and Flask.

## Features

- NVD CVE Collection
- CVE Report Generation
- CVSS Risk Classification
- Security Advisory Templates
- REST API
- Web Dashboard
- Severity Tracking

## Technology Stack

- Python
- Flask
- JSON
- Bootstrap 5

## API

GET /api/cves

## Run Locally

pip install -r requirements.txt
python app.py

Open:
http://127.0.0.1:5000
