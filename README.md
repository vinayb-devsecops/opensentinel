# OpenSentinel

Autonomous Vulnerability Intelligence Platform built with Python and Flask.

## Features

- NVD CVE Collection
- CVE Report Generation
- CVSS Risk Classification
- Security Advisory Templates
- REST API
- Web Dashboard
- CVE Search

## Technology Stack

- Python
- Flask
- Bootstrap 5
- JSON

## API Endpoints

- /api/cves
- /api/cves/<cve_id>
- /api/stats

## Architecture

Collector -> Processing -> Risk Engine -> API -> Dashboard
